# Template for building your own microservice

This is a template for user to refer when creating their own microservice. It includes c++/Java/python 3 versions of microservice implementation. You can choose your favorite language and implement your own service based on the template.

## Major Dependencies

- [Facebook Thrift](https://github.com/facebook/fbthrift)
- [gradle](https://gradle.org/)
